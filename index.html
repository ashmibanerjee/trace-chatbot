TRACE: A Conversational Framework for Sustainable Tourism Recommendation with Agentic Counterfactual Explanations

Ashmi Banerjee1 , Adithi Satish1, Wolfgang Wörndl1, and Yashar Deldjoo2
1*Technical University of Munich, Germany 2Polytechnic University of Bari, Italy
Paper arXiv Demo Code

Abstract



Many conversational travel recommender systems focus almost exclusively on immediate user relevance and convenience. While effective, this emphasis can unintentionally encourage travel to overcrowded destinations and promote carbon-intensive choices.

To address this gap we propose TRACE (Tourism Recommendation with Agentic Counterfactual Explanations), a modular, multi-agent, LLM-based framework that nudges users toward more sustainable options while preserving recommendation quality. TRACE uses an orchestrator–worker architecture in which specialized agents elicit latent sustainability preferences, build concise user personas, and generate recommendations that balance personal relevance with environmental impact.

A key contribution of TRACE is its use of agentic counterfactual explanations: lightweight, transparent suggestions that present greener alternatives and encourage reflection instead of coercion. We implement TRACE using Google’s Agent Development Kit and evaluate the approach through a user study and complementary semantic-alignment analyses, showing that TRACE can support more sustainable decision-making without degrading recommendation utility.
